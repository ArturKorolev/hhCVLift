# CVLift

Расширение для браузера для автоматического поднятия резюме на hh.ru.

## Что делает

- Автоматически поднимает резюме на сайте hh.ru по расписанию
- Работает как браузерное расширение для Chrome/Edge
- Предоставляет простой интерфейс для управления настройками

## Установка

1. Откройте страницу расширений браузера.
2. Включите режим разработчика.
3. Нажмите "Загрузить распакованное расширение".
4. Выберите папку проекта.

## Структура проекта

- `background.js` — фоновый скрипт и логика поднятия резюме
- `content.js` — взаимодействие со страницей hh.ru
- `popup.html` / `popup.js` / `popup.css` — интерфейс расширения
- `manifest.json` — настройки расширения

## Примечание

Это учебный/личный проект для автоматизации рутинных действий на hh.ru.

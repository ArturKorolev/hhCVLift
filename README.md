# HH Resume Lifter

Расширение для Chrome/Edge, которое помогает поднять резюме на hh.ru автоматически.

## Как работает

- В расширении задаётся ссылка на резюме hh.ru.
- Нажимаешь кнопку `Включить автоподнятие на 7 дней`.
- Расширение автоматически проверяет резюме и пытается нажать кнопку `Поднять` сразу при включении, а затем примерно каждые 4 часа и 1 минуту.
- Через 7 дней расширение автоматически выключается.

## Установка

1. Открой `chrome://extensions/` или `edge://extensions/`.
2. Включи `Режим разработчика`.
3. Нажми `Загрузить распакованное расширение`.
4. Выбери папку этого проекта.

## Использование

1. Авторизуйся на hh.ru.
2. Открой расширение и вставь ссылку на резюме.
3. Нажми `Включить автоподнятие на 7 дней`.
4. Если нужно, нажми `Остановить`.

## Файлы

- `manifest.json` — конфигурация расширения.
- `popup.html` / `popup.js` — интерфейс расширения.
- `background.js` — фон, который запускает проверки.
- `contentScript.js` — скрипт, который кликает кнопку на hh.ru.

## Примечание

Если hh.ru поменяет текст кнопки или страницу, расширение может не сработать. Тогда нужно обновить селектор в `contentScript.js`.
